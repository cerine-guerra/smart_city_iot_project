## 🌆 Mini Smart City Project – IoT with ESP32 & Blynk

A **Mini Smart City simulation** using ESP32, Blynk, and various sensors to demonstrate a connected city environment and IoT concepts in action.

---

### 🔹 Project Overview
- **ESP32 Microcontroller** controls sensors and actuators.  
- **Blynk IoT Platform** enables remote monitoring and control.  
- **Sensors & Devices:** DHT11, MQ2 Gas Sensor, Flame Sensor, IR Sensor, Servo Motor, LCD Display, Buzzer.

**Automation Features:**
- Real-time temperature & humidity monitoring  
- Gas leak and flame detection with alerts  
- Automated gate using IR sensor and servo  
- Traffic light system simulation  

---

### 🔹 Tools & Technologies
- Arduino IDE  
- ESP32  
- Blynk IoT App  
- Sensors: DHT11, MQ2, Flame, IR  
- Actuators: Servo Motor, LCD Display, Buzzer  

---

### 🔹 Features
- Real-time environmental monitoring  
- Automated hazard response (gas, flame, temperature)  
- Remote control and notifications via Blynk  
- Mini smart city simulation for testing IoT concepts  

---

### 🔹 How to Run
1. Install **Arduino IDE** and necessary libraries (`Blynk`, `DHTesp`, `ESP32Servo`, etc.)  
2. Connect **ESP32** to your computer and upload the `.ino` code  
3. Set your **WiFi credentials** in the code  
4. Open the **Blynk app** and connect using the provided auth token  
5. Monitor sensors and control the system remotely
