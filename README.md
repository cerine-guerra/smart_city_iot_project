# smart_city_iot_project
This project is a Mini Smart City simulation using ESP32, Blynk, and various sensors to demonstrate a connected city environment.
Project Overview
ESP32 Microcontroller controls sensors and actuators.
Blynk IoT Platform for remote monitoring and control.
Sensors & Devices: DHT11, MQ2 Gas Sensor, Flame Sensor, IR Sensor, Servo Motor, LCD Display, Buzzer.
Automation Features:
Temperature & humidity monitoring
Gas leak and flame detection with alerts
Automated gate using IR sensor and servo
Traffic light system simulation
Tools & Technologies
Arduino IDE
ESP32
Blynk IoT App
DHT11, MQ2, Flame, IR sensors
Servo Motor, LCD Display, Buzzer
Features
Real-time environmental monitoring
Automated hazard response (gas, flame, temperature)
Remote control and notifications via Blynk
Mini smart city simulation for testing IoT concepts
How to Run
Install Arduino IDE and necessary libraries (Blynk, DHTesp, ESP32Servo, etc.)
Connect ESP32 to your computer and upload the .ino code.
Set your WiFi credentials in the code.
Open the Blynk app and connect using the provided auth token.
Monitor sensors and control the system remotely.
